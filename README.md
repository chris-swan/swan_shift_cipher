# Swan Cipher

##### A Drupal module for encode an input.

#### By Chris Swan

## Description
An app that changes the letters of a word by a given value up or down.

## Setup

#### LOGIN INFORMATION


Database
User: epicodus
Password: root

Drupal
User: cipher
Password: root

#### INSTRUCTIONS FOR RUNNING THIS PROJECT

 1. Clone repository.
 2. Direct your favorite Apache server to the repository as document root and start the server.
 3. Open PHPMyAdmin and import the database found in /sites/backup-db
 4. Direct your browser to localhost
